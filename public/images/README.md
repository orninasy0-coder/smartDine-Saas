# Images Folder

## Chef 3D Image

To add the chef image to the Smart Kitchen Partner section:

1. **Download the image** from Freepik:
   - Go to: https://www.freepik.com/premium-psd/3d-cartoon-chef-closeup-halfbody-isolated-transparent-background-png-psd_60432695.htm
   - Download the PNG version (with transparent background)

2. **Save the image** in this folder as:
   - `chef-3d.png`

3. **Image Requirements**:
   - Format: PNG (with transparent background preferred)
   - Recommended size: 500x500px or larger
   - The image will be automatically scaled to fit

4. **Alternative Images**:
   - If you want to use a different chef image, just replace `chef-3d.png`
   - Make sure it has a transparent background for best results
